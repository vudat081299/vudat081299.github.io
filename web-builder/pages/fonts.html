Đổi font toàn hệ — sửa 1 token

Mọi component đọc --wb-font, nên đổi 1 dòng là đổi cả app. Ví dụ chuyển sang Inter:

/* 1. Nạp font (index.html hoặc app của bạn) */
<link rel="stylesheet"
  href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap">

/* 2. Trỏ token về font mới — trong web-builder.css hoặc CSS app */
:root {
  --wb-font: "Inter", -apple-system, BlinkMacSystemFont, "Segoe UI",
               Roboto, Helvetica, Arial, sans-serif;
}
/* Xong. Toàn bộ button/table/capsule/số… tự đổi theo. */
💡
Trong app của bạn (Vite), nên tự host font (vd gói @fontsource/inter, @fontsource/jetbrains-mono) thay vì gọi Google Fonts để nhanh & ổn định. Bật font-feature-settings: "tnum" 1 cho vùng số nếu font hỗ trợ.
